# 1. 购物街

一个移动端商城项目，实现了商城主页和商品推荐、商品详情页和购物车。

项目基于vue+router+vuex+axios，通过vue实现数据驱动，减少了最麻烦的dom操作和组件复用，提高项目开发效率，vuex其中管理用户和购物车等数据流。

## 1.1 安装项目依赖

```
npm install
```
## 1.2 编译并运行devServer
```
npm run dev
```
## 1.3  启动数据模拟server
```
npm run server
```
